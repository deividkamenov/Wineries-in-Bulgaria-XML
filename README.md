## XML 2023 – Project Wine Catalogue

Small XML/XSLT project that models a **wine & winery catalogue** and demonstrates:

- XML data modelling
- DTD validation
- XSLT transformation to a printable / PDF‑ready layout
- Working with images and assets

### Contents

- `XML/catalogue.xml` – main XML data file (wine & winery catalogue)
- `XML/catalogueDTD.dtd` – DTD defining the XML structure and constraints
- `XML/catalogueXSLTToPDF.xsl` – XSLT stylesheet used to format the catalogue
- `XML/wineries.jpg`, `XML/winery_image*.jpg`, `XML/wine_image*.jpg` – images used in the catalogue
- `XML/catalogue.pdf` – generated PDF version of the catalogue
- `XML/XML_Project2023.xpr` – project file from the XML authoring tool
- `XML/XML_5_62585_62645_.docx` / `.pdf` – project documentation (Bulgarian)

IDE‑specific folders (`.idea`) and OS metadata (`__MACOSX`, `.DS_Store`) have been removed so the repository stays clean and portable.

### How to view / transform

- Open `catalogue.xml` together with `catalogueXSLTToPDF.xsl` in an XML/XSLT‑capable editor (e.g. Oxygen XML).
- Run the XSLT transformation to produce a formatted document or PDF (depending on your tool configuration).
- Alternatively, open the already generated `catalogue.pdf`.

### Portfolio focus

This project is suitable for showcasing:

- XML data modelling and validation with DTD
- XSLT styling for printable catalogues
- Handling of multi‑file XML projects with images and documentation
